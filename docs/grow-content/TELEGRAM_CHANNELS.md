## Скрипт мониторинга — триггерные фразы

Ищи в поиске Telegram и отвечай на треды с такими фразами:

```
"сколько брать за"
"какую ставку"
"как посчитать ставку"
"почасовка"
"как считать час"
"не знаю сколько стоит"
"сколько стоит час"
"как поднять ставку"
"сколько просить"
"минимальная ставка"
"самозанятый налог ставка"
```

## Правила размещения

1. **Не спамить** — одно сообщение в один чат максимум раз в месяц
2. **Отвечать на реальные вопросы** — только если тема уже обсуждается
3. **Не копировать один пост во все чаты** — адаптируй под аудиторию
4. **Выжидать** — в новом чате сначала поучаствуй в паре обсуждений без ссылки
5. **Модераторы** — в крупных чатах (@frilanc и т.п.) лучше написать модератору заранее
